## ✨ What is this?

**NaPa Counter** is a sleek, high-performing web application designed for those who want to track their "NaPa" (or any pill) intake with absolute style. Forget boring spreadsheets—this is counting with **flavor**.

### 🧪 Features that Pop:

- 🖋️ **Bengali Numerals:** Native digit support for that authentic feel.
- 🌈 **Dynamic Progress:** A progress bar that shifts from **Resting Red** to **Emerald Success** as you reach your goal.
- ⚡ **Turbo Navigation:** Smooth, hardware-accelerated transitions powered by **Framer Motion**.
- ⌨️ **God Mode (Keyboard):**
    - `Space` to Tap 👆
    - `Enter`/`Space` to Start/Restart 🔄
    - `Escape` to Abort 💨
- 💾 **No Amneisa:** Mistaken reloads? No problem. **LocalStorage** persistence keeps your session alive.
- 📏 **Auto-Focus:** Land on the page and start typing immediately. Your cursor is already where it needs to be.
